🍕 Pizza Sales Data Analysis – Business Insights Dashboard
📘 Overview

This project analyzes a Pizza Sales dataset to uncover actionable business insights using SQL and Power BI.
It focuses on building an interactive dashboard that highlights key KPIs, sales trends, and Top 5 / Bottom 5 performance analysis, enabling data-driven decision-making for business stakeholders.

⚙️ Project Workflow

Data Understanding & Preparation – Reviewed dataset structure, cleaned inconsistencies, and standardized fields for analysis.

SQL-Based Analysis – Executed SQL queries to calculate KPIs and extract insights related to sales, orders, categories, and product performance.

Data Modeling – Structured data for efficient reporting and accurate aggregations in Power BI.

Power BI Dashboard Development – Designed an interactive dashboard with filters, visuals, and KPI cards for intuitive exploration.

🧰 Tech Stack

Languages: SQL
Visualization Tool: Power BI
Database: Relational sales dataset

📊 Key Insights

Identified key KPIs such as total revenue, total orders, average order value, and pizzas sold

Analyzed Top 5 and Bottom 5 pizzas based on revenue, quantity, and orders

Explored sales trends by time, category, and size

Highlighted best-performing and underperforming products for strategic planning

🚀 Outcome

Delivered an end-to-end sales analytics dashboard using SQL and Power BI that transforms raw pizza sales data into meaningful business insights, supporting performance monitoring and strategic decision-making.
